Knapsack Problem No number limit
----

重さと価値がそれぞれwi, viであるようなn種類の品物があります。
これらの品物から、重さの総和がWを超えないように選んだときの、価値の総和の最大値を求めなさい。
ただし、同じ種類の品物をいくつでも選ぶことができます。

制約

* 1 <= n <= 100
* 1 <= wi, vi <= 100
* 1 <= W <= 10000

## 例

    n = 3
    (w, v) = {(3, 4), (4, 5), (2, 3)}
    W = 7

が与えられた時、

    10

が答えとなる。
