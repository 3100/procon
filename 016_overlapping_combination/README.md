Overlapping combination Problem
----

n種類の品物があり、i番目の品物はai個あります。
異なる種類の品物同士は区別できますが、同じ種類の品物同士は区別できません。
これらの品物の中からm個選ぶ組み合わせの総数を求め、Mで割った余りを答えなさい。

制約

* 1 <= n <= 1000
* 1 <= m <= 1000
* 1 <= ai <= 1000
* 2 <= M <= 10000

## 例

    n = 3
    m = 3
    a = {1, 2, 3}
    M = 10000

が与えられた時、

    6(0+0+3, 0+1+2, 0+2+1, 1+0+2, 1+1+1, 1+2+0)

が答えとなる。
